In my libary there are colors to draw sprites with it here are colors
0 - white
1 - black
2 - red
3 - green
4 - blue
5 - yellow
6 - purple
7 - cyan
8 - gray
9 - orange
10 - pink
11 - skin color
12 - brown

if you want to not draw that part you must just write ' ', but there is sprite gallery to chose that sprite that you want

To create sprite you need to write let sprite = CWS.createSprite(img, size, x, y, rotation)

IMG must be an array like this [
[' ', 2, ' ', 2, ' '],
[2, 2, 2, 2, 2],
[2, 2, 2, 2, 2],
[' ', 2, 2, 2, ' '],
[' ', ' ', 2, ' ', ' ']
].

Size is the every pixel size, its default 20

x and y are default 0

rotation is for rotating sprite its default 0

remember that you create sprite, but you don't draw it

To draw the Sprite you need to write CWS.drawSprite(sprite, canvas)

You dont give the context in the canvas you just give the canvas.

Here is example
let canvas = document.querySelector("canvas")
let sprite = CWS.createSprite([
[' ', 2, ' ', 2, ' '],
[2, 2, 2, 2, 2],
[2, 2, 2, 2, 2],
[' ', 2, 2, 2, ' '],
[' ', ' ', 2, ' ', ' ']
]) CWS.drawSprite(sprite, canvas)

To draw some sprites you need to write CWS.drawSprites(sprites, canvas)

Sprites must be array like this [sprite1, sprite2, sprite3, ...]
To move sprite you need to write CWS.moveSprite(sprite, x, y, rotation)

x is like margin left, and y like margin top

If you want that the sprite move forever like the rocket at the right, you need to use requestAnimationFrame
When you are using request animation frame, you need allways to clean the canvas, to clean canvas you need to use CWS.clear(canvas)
CWS.isCollision(sprite1, sprite2) returns true or false. You can use it like this

if(CWS.isCollision(sprite1, sprite2)){
alert("bam")
}
CWS.isInCanvas(sprite, canvas) returns true or false. You can use it like this

if(!CWS.isInCanvas(sprite, canvas)){
alert("you are out of board")
}
CWS.setPosition(sprite, x, y) this is for changing the position
x and y are not adding like moveSprite, they are changing.
CWS.centerSprite(sprite, canvas) is for placing sprite at the midel of canvas by x and y
CWS.centerSpriteByX(sprite, canvas) is for placing sprite at the midel of canvas by x
CWS.centerSpriteByY(sprite, canvas) is for placing sprite at the midel of canvas by y
CWS.cloneSprite(sprite, x, y, size, rotation) is for cloning sprite, and you can change rotation, size
CWS.followSprite(enemy, player, enamySpeed) is for sprite1 "enemy" to follow sprite2 "player"

The speed is default 1
CWS.keepInCanvas(sprite, canvas) is don't letting the sprite come out of canvas

You must write that in request animation frame
CWS.arrowControl(sprite, speed) is for controling sprite with arrow up, arrow down, arrow left and arrow down.

You dont type it in request animation frame, but you must have request animation frame
CWS.WASDControl(sprite, speed) is for controling sprite with w, a, s and d keys.

You dont type it in request animation frame, but you must have request animation frame
